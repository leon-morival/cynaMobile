# Cyna Mobile

## Installation

```bash
npm i
```

## Développement

Utilisez le build de développement :

1. Préparez le build

   ```bash
   npx expo prebuild --clean
   ```

2. Lancez sur l'appareil/émulateur

   ```bash
   # Pour Android
   npx expo run:android

   # Pour iOS
   npx expo run:ios
   ```
